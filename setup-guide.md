# Panduan Setup FROST77 - Website Slot Online Mutakhir

FROST77 adalah website slot online dengan fitur 3D, AI, dan keamanan tinggi. Ikuti langkah-langkah ini untuk membangun proyek dari awal.

## Prasyarat
- **Node.js** (v16 atau lebih baru): Unduh dari nodejs.org.
- **MongoDB Atlas**: Daftar di mongodb.com/cloud/atlas.
- **Blender**: Untuk membuat aset 3D (opsional).
- **Figma**: Untuk desain UI.
- **Keycloak**: Untuk autentikasi SSO.
- **Cloudflare**: Untuk keamanan DDoS.
- **Vercel dan AWS CLI**: Untuk deployment.
